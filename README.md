# Youtube Downloader App
## introduction
dist/main.exe  
main.exeを起動するだけで使用することができる  
音声のみをDLする場合は、ffmpegがインストールされていないとエラーが起きる  
ffmpegをインストールするには、  
[ffmpeg公式サイト](https://www.ffmpeg.org/download.html)
